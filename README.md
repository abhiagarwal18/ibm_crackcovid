# TEAM REGRESSION
 
 IBM Crack the COVID Hackathon


**Dependencies**

`npm install`   -> To install all the dependecies.

- express
- express-session
- passport
- ibmcloud-appid

**Usage**

`node.exe app.js`

Then open [localhost:3000](http://localhost:3000) in your browser.
